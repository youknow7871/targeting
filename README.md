# CRM AI Campaign Architect (Targeting)

A premium, AI-powered web tool for recommending optimal CRM campaign content based on historical success data.

## Features

- **AI-Driven Recommendations**: Suggests high-performing campaign titles and content.
- **KPI Prediction**: Predicts CTR and CVR improvement based on historical benchmarks.
- **Keyword Lift Analysis**: Visualizes keywords that statistically improve campaign performance.
- **Premium UI**: Dark mode, glassmorphism, and smooth interactive elements.

## Tech Stack

- HTML5 / Vanilla CSS
- Pure JavaScript (ES6+)
- Data-driven Recommendation Engine

## Usage

1. Open `index.html` in any modern browser.
2. Select campaign parameters (Segment, Time, Medium, etc.).
3. Click "AI 추천 생성하기" (Generate AI Recommendations).
4. Review high-probability success templates and copy them for use.
